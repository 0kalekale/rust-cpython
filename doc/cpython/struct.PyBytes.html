pub struct PyBytes(_);

Represents a Python byte string. Corresponds to str in Python 2, and bytes in Python 3.

fn extract<'a>(py: Python, o: &'a PyObject) -> PyResult<Cow<'a, str>>

Extract a rust string from the Python object.

In Python 2.7, accepts both byte strings and unicode strings. Byte strings will be decoded using UTF-8.

In Python 3.x, accepts unicode strings only.

Returns TypeError if the input is not one of the accepted types. Returns UnicodeDecodeError if the input is not valid unicode.
