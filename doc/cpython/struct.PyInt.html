pub struct PyInt<'p>(_);

Represents a Python int object.

Note that in Python 2.x, int and long are different types. When rust-cpython is compiled for Python 3.x, PyInt and PyLong are aliases for the same type, which corresponds to a Python int.

You can usually avoid directly working with this type by using ToPyObject and extract with the primitive Rust integer types.
